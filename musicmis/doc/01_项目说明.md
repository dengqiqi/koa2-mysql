# i音悦网站后台管理系统
## 项目说明

### 项目准备

>* 配置
>1. 安装node7.6版本以上，配置环境变量
>2. 按照package.json文件中的安装相关依赖<br/>
>  windows下使用npm 命令进行安装，建议安装使用cnpm，linux下使用sudo命令
>3. 使用git进行版本控制
>4. 使用mysql数据库
>* 构建项目目录如下

### 项目目录


```
.
├── db                    // 编  程：封装mysql表对象
│   ├── adminDb.js
│   ├── databaseConnection.js

```
